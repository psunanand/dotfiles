# MK's dotfiles

## Stack

- **Shell:** Zsh (antidote, fzf-tab, zsh-vi-mode, autosuggestions, syntax highlighting)
- **Editor:** Neovim
- **Terminal:** Kitty & Tmux
- **WM:** Aerospace & Jankyborders
- **Bar:** Sketchybar
- **Packages:** Homebrew

## Zsh highlights

- **Nav:** zoxide, AUTO_CD, AUTO_PUSHD
- **Completion:** FZF + fzf-tab, case-insensitive matching, eza previews
- **History:** incremental append, cross-shell share, dup/space filtering
- **Safety:** interactive cp/mv, CORRECT, RM_STAR_WAIT
- **Integrations:** Kitty, Homebrew command-not-found
- **Tools:** bat, eza, fd, ripgrep, delta, rip
